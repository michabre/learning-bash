# Linux Processes

## Commands

```shell
# show currently running processes
ps

# show prcoesses run by a specific user
ps -u <user id>

# kill process
kill <process or job id>

# get process id
pgrep <process>

# show running processes sorted by cup usage
top

# pretty way to show running processes
htop

# force end a process
kill -9 <process>

# see all kill options
kill -l

# stop process
kill -19 <process id>

#kill process
kill -9 <process id>

# kill a process by name
pkill -9 <name of process>
```
